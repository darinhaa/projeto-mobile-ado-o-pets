# Introdução
O Aumigo é um site criado para facilitar a adoção de animais, conectando pessoas a pets que precisam de um novo lar. Ele oferece uma navegação simples e acolhedora, permitindo que o usuário conheça os animais disponíveis e leia suas histórias antes de iniciar o processo de adoção.

Para sua criação, foi utilizado React Native, aliado a um framework baseado em HTML e CSS, permitindo a construção de interfaces modernas, responsivas e compatíveis com dispositivos móveis. O uso dessas tecnologias garante uma experiência fluida, visualmente agradável e fácil de usar, refletindo o propósito central do projeto: aproximar pessoas e pets de forma humanizada e eficiente.

# Tela inicial
A tela inicial apresenta um banner com a imagem de vários pets, reforçando o propósito do site. Logo abaixo, há um botão chamativo “Quero adotar!”, que leva o usuário para a área de listagem dos animais disponíveis.

A página também traz duas pequenas seções informativas: a primeira convida o usuário a encontrar seu novo animal, explicando que o site reúne diversos pets esperando por um lar; a segunda explica brevemente por que adotar é importante, destacando o impacto positivo na vida dos animais e incentivando uma adoção responsável.

# Lista de pets para adoção
Essa tela mostra os animais disponíveis para adoção organizados em cartões.

Cada cartão exibe a foto do pet, seu nome, localização e algumas informações essenciais, como espécie e idade.

É uma tela pensada para navegação rápida: o usuário pode visualizar todos os animais de forma clara e escolher qual deseja conhecer melhor. Ao clicar em um dos pets, o usuário é direcionado para a página de detalhes.

# Detalhes do pet selecionado
Quando o usuário escolhe um animal, essa tela exibe informações completas sobre ele.

No topo, aparece uma imagem maior do pet, seguida de seu nome, idade, espécie, gênero e porte. Logo abaixo, a localização é mostrada de maneira mais precisa, incluindo o bairro.

Também há a data em que o animal foi resgatado ou disponibilizado para adoção e um texto contando sua história de forma carinhosa e detalhada, ajudando o usuário a criar conexão emocional.

Por fim, a página oferece dois botões principais: um incentivo para mudar a vida do animal e o botão “Quero adotar!”, que representa a ação final que o site deseja que o usuário tome.
